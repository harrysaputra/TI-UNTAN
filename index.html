TI-UNTAN by harrysaputra

account untuk ujicoba penggunaan git bagi dosen/mahasiswa/alumni teknik informatika untan pontianak

Halo teman2 TI UNTAN , silahkan praktek penggunaan Git di repository ini yah, selengkapnya kita diskusikan di facebook fans group. Intinya di repository ini kalian bisa melakukan apapun yang berhubungan dengan Git seperti commit, add, pull, push, create branch, merge, revert dan lainnya. Praktek ya :) , ini penting sekali lo, karena sudah standar di software house / company dan rata-rata freelancer juga bekerja dengan sistem ini supaya mudah berkolaborasi !
